My Portfolio

Quiz App

I am currently working on a quiz application which was inspired by my role as a teacher. The quiz application allows you to import a csv file, which is then processed and outputs a quiz that will also create flashcards from the questions you got incorrect. You can also then reattempt those questions to see if you get them right. Currently the application is only command-line based but will be migrated to a GUI to make it more user-friendly, especially for younger people.
